# Nexo Studio 🚀

**Nexo Studio** es un generador de micro-sitios Bio-Link autónomos, altamente estéticos y profesionales. Diseñado para creadores que buscan una presencia digital premium con la simplicidad de un archivo HTML único y portable.

## ✨ Características Principales

- **Dashboard Glassmorphism**: Interfaz de administración ultra-moderna con efectos de desenfoque y neón.
- **Live Preview**: Visualización en tiempo real dentro de un mockup de smartphone.
- **Export Engine Pro**: Genera archivos HTML 100% autónomos con imágenes embebidas en **Base64**.
- **Personalización Extrema**:
  - Modos de fondo: Sólido, Degradado, Mesh Gradient y Multimedia.
  - Geometría de botones: Pastilla (Pill), Redondeado y Cuadrado.
  - Estilos de botones: Cristal (Glass), Sólido y Contorno.
  - Tipografías modernas integradas.
- **Social Auto-Detect**: Detección automática de iconos para Instagram, YouTube, TikTok, X (Twitter), LinkedIn, Spotify y más.

## 🛠️ Stack Tecnológico

- **Backend**: Node.js + Express.js.
- **Frontend**: EJS (Templates), CSS3 Moderno (Glassmorphism), Vanilla JavaScript.
- **Seguridad**: Helmet.js con configuración CSP personalizada.
- **Utilidades**: Axios (Procesamiento de imágenes), FontAwesome 6.

## 🚀 Instalación y Uso

1. **Clonar el repositorio**:

   ```bash
   git clone https://github.com/AndresSY26/Nexo-Studio.git
   cd Nexo-Studio
   ```

2. **Instalar dependencias**:

   ```bash
   npm install
   ```

3. **Iniciar en modo desarrollo**:

   ```bash
   npm run dev
   ```

4. **Acceder al Dashboard**:
   Abre [http://localhost:3000](http://localhost:3000) en tu navegador.
